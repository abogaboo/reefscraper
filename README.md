requires a repo.xlsx workbook and a main.xlsx workbook on the user's computer

scrape_all(district) takes a string district_name as an argument and scrapes data from that district:
0. Chesapeake
1. Michigan
2. Indiana
3. Texas
4. Mid-Atlantic
5. North Carolina
6. South Carolina
7. Israel
8. New England
9. Ontario
10. Peachtree
11. Pacific NW

scrape_all_districts() scrapes data for every district except the regional pool.
